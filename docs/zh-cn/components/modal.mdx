| 属性                | 描述           | 类型             | 可选值             | 默认     |
| ------------------- | -------------- | ---------------- | ------------------ | -------- |
| **v-model:visible** | 显示与隐藏     | `Boolean`        | `'true','false'`   | `false`  |
| **title**           | 标题           | `string`         | `-`                | `-`      |
| **width**           | 对话框宽度     | `string`         | `-`                | `420px`  |
| **cancel**          | 取消按钮的文字 | `string`         | `-`                | `cancel` |
| **done**            | 确定按钮的文字 | `string`         | `-`                | `done`   |
| ...                 | 原生属性       | `HTMLAttributes` | `'class','id',...` | `-`      |

</fe-Attributes>
